Mas o que é uma API?

API vem do inglês e significa 'Application Programming Interface' ('Interface de Programação de Aplicações'), que especifica como componentes de um software devem interagir uns com os outros. No nosso contexto, isso significa que vamos retornar diferentes conjuntos de dados para os programadores ou interessados usarem, permitindo seu uso e manipulação em suas próprias aplicações.

Este serviço, do Gastos Abertos, cria diversas oportunidades para quem não necessariamente tem conhecimento, tempo e recursos para pegar todos os dados do orçamento sozinho, mas está cheio de ideias sobre como usá-los. Então, se você tem uma ideia que precisa apenas dos dados orçamentários para tornar-se realidade, não hesite em nos contatar ou até mesmo contribuir com nosso repositório no GitHub. Queremos que todos os dados orçamentários do país tenham uma API!
